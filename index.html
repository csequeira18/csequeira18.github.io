Birthday Paradox

The birthday paradox says that the probability that two people in a room will have the same birthday is more than half, provided n, the number of people in the room, is more than 23.

This project runs a simulation of the birthday paradox. One simulation involves running experiments testing the paradox on trial groups ranging from 5 to 100 people. In an experiment, 20 trials will be run on the given number of people being tested on. Random birthdays are generated for each trial run, and the amount of duplicate birthdays is tracked per exepr

I took charge of writing the Trial class, as well as the Experiment class. Throughout the project we focused on implementing clean code practices as well as clear commit messages.
